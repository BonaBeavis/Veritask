# Installation

## Development
Install dependencies:
* git
* sbt

```bash
git clone https://github.com/BonaBeavis/Veritask.git
cd Veritask
make start-dev
```

## Staging
Install dependencies:
* git
* make
* docker
* docker-compose

```bash
git clone https://github.com/BonaBeavis/Veritask.git
cd Veritask
make start-staging
```
Admin frontend is reachable under localhost:9000.
