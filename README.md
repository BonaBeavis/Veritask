= Development =
Install dependencies:
* git
* sbt

```bash
git clone https://github.com/BonarBeavis/Veritask.git
cd Veritask
make start-dev
```

Admin frontend is reachable under localhost:9000.

= Deployment =

== Staging ==
Install dependencies:
* git
* make
* docker
* docker-compose

```bash
git clone https://github.com/BonarBeavis/Veritask.git
cd Veritask
make start-staging
```
